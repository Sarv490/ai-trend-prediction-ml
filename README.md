# AI Tools Trend Prediction Using Machine Learning

## Overview
This project predicts the popularity trends of AI tools such as ChatGPT, Gemini, and Meta AI using Google Trends data and Machine Learning forecasting techniques.

## Features
- Data Collection using Google Trends
- Exploratory Data Analysis (EDA)
- Trend Visualization
- Time-Series Forecasting
- Model Performance Comparison
- Future Popularity Prediction
- Interactive Streamlit Dashboard

## Models Used
- Random Forest Regressor
- Gradient Boosting Regressor
- Naive Forecasting

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Streamlit

## Results
The Naive Forecasting model achieved the best performance because the dataset exhibited relatively stable trend behaviour while more complex models showed signs of overfitting.

## Repository Structure
- app.py → Streamlit Dashboard
- AI_Trend_Prediction.ipynb → Model Training and Analysis
- actual_vs_predicted.png → Prediction Results
- ml_comparison.png → Model Comparison
- 12_week_prediction.csv → Future Forecast
